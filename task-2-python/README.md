# Task 2 - Python

Dieses Projekt demonstriert **Formatting**, **Linting** und **Testing** in Python.  
Bevor die Applikation gestartet wird, werden automatisch:
1. Der Code formatiert (Black)
2. Der Code gelinted (Flake8)
3. Die Tests ausgeführt (Pytest)

Nur wenn alle Schritte erfolgreich sind, wird die Applikation gestartet.

---

## Voraussetzungen

- Installiertes [Python 3.x](https://www.python.org/downloads/)
- `pip` für Paketinstallation
